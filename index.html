Installing WordPress

Before beginning the installation and configuring of the plugin, you must first have WordPress already installed on a server. Here is a very detailed guide at Wordpress Codex that will walk you through every step.

Installing Plugin

Uploading Through WordPress

To upload through your Dashboard, select “Plugin” . Next, select the “Add New” button at the top of the page and then select the “Upload Plugin” button. Choose Lordiconizer - Animated Icons WP Plugin for Elementor and WPBackery Plugin "lizr.zip" file and select “Install Now”. Once the plugin is fully uploaded and installed, click “Activate” to activate the WordPress plugin.

When uploading your plugin with the installer, please ensure you are uploading the plugin .zip file, not the entire package you downloaded. In this case, you will be uploading "lizr.zip".

FTP Upload

To manually upload your new WordPress plugin, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the un-zipped Lordiconizer - Animated Icons WP Plugin for Elementor and WPBackery Plugin folder into the: wp-content/plugins folder.
Once uploaded, activate the plugin by heading to the “Installed Plugins” menu in the WordPress Dashboard.

Update Plugin

Update via Envato Market Plugin

To easy and fast update plugins you can use Envato Market Plugin :
1) Install Envato Market Plugin: https://envato.com/market-plugin/ and activate.
2) On admin panel go to Envato Market -> Settings -> Generate an Envato API Personal Token -> Click on link -> Login to your Envato Account -> Checked “Permissions ( https://prnt.sc/TAWhHWx09CtK )” on list -> Create Token.
3) Use plugin “Envato Market -> Plugins” to install all future updates. (in a few clicks).
Thanks!

Manually Update (Plugins)

You can update our plugin via WordPress Panel, Go to Dashboard > Plugins . Click "Add New" -> "Upload Plugin". Choose Lordiconizer - Animated Icons WP Plugin for Elementor and WPBackery Plugin "lizr.zip" file and select “Install Now”.

Manually Update (FTP or Cpanel)

You can update our plugin using via FTP or Cpanel.
Download the latest version of the plugin on Codecanyon download page and you rename the folder contain the old plugin and create a new folder and insert the latest plugin in here.
