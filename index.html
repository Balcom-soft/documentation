Bulk User Generator - Create Multiple User Accounts for WrodPress at a Single Click

Thank you for purchasing our plugin!

Bulk User Generator plugin allows you to generate multiple new users with any available role at a single click. Comma separated email addresses can be used to create new user account. Even if you don't provide any data, this plugin can generate random unique fake email addresses for your users.

The plugin keep all records of generated users and you can observe the detailed report at any time. Generated user account cridentials can be downloaded in a plain text format. You can distribute the login cridentials among your employees, students or any group of people you want with ease. This keeps you from going through the hassle of registration process one by one to generate multiple user account.

We have taken a good care of the security. You have the full control over your generated user accounts. Only the user cridentials created by the plugin can be viewed by the admins. The plugin does not interfere with your website's natural registration process.

Installing WordPress

Before beginning the installation and configuring of the plugin, you must first have WordPress already installed on a server. Here is a very detailed guide at Wordpress Codex that will walk you through every step.

Installing Plugin

Uploading Through WordPress

To upload through your Dashboard, select “Plugin” . Next, select the “Add New” button at the top of the page and then select the “Upload Plugin” button. Choose Bulk User Generator plugin "bulk-user-generator.zip" file and select “Install Now”. Once the plugin is fully uploaded and installed, click “Activate” to activate the WordPress plugin.

When uploading your plugin with the installer, please ensure you are uploading the plugin .zip file, not the entire package you downloaded. In this case, you will be uploading "bulk-user-generator.zip".

FTP Upload

To manually upload your new WordPress plugin, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the unzipped Bulk User Generator plugin folder into the: wp-content/plugins folder.
Once uploaded, activate the plugin by heading to the “Installed Plugins” menu in the WordPress Dashboard.

Generating Users

To start generating new user accounts navigate to the “Generate new” page under the standard WordPress Users admin menu:



There are two ways to create new user accounts: using comma separated emails addresses list or generating unique fake email addresses.

Provide comma separated emails list to generate new users for each email address. The plugin skips any duplicate emails, so don't worry about emails that already have accounts on your website. You can can assign a user role for the accounts to be generated. Random first names and last names can also be generated for each new account.

Enabling welcome emails sends notifications to new user emails. Please, note that this feature is only available for CSV option with real email addresses. When using fake emails you will not be able to send welcome notifications.

Selecting Fake emails option enables you to create new user account by generating unique fake email addresses. Accounts with fake email addresses are no different from the real ones except they use non-existent email addresses. You have to specify the number to indicate the number of accounts to be generated.

Viewing Reports

Each time you generate users, the plugin records a detailed report that contain all the information you need about the process and the accounts. On the index page you can observe quick stats about the batch user generate job. Using button on the right side of the table you can download detailed reports and get login cridentials.

Detailed report shows extended view of everything that happened during the user generate process. You can find information about any errors or skipped tasks there.

Account cridentials are exported in a plain text format. You can easily distribute username and passwords among your students, employees or colleagues.
