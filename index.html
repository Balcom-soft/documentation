Installing WordPress

Before beginning the installation and configuring of your new theme, you must first have WordPress already installed on a server. Here is a very detailed guide at WordPress Codex that will walk you through every step.

Installing Theme

Uploading Through WordPress

To upload through your Dashboard, select “Appearance” and then “Themes”. Next, select the “Add New” button at the top of the page and then select the “Upload Theme” button. Choose Tenpo theme "tenpo.zip" file and select “Install Now”. Once the theme is fully uploaded and installed, click “Activate” to activate the WordPress theme.

When uploading your theme with the installer, please ensure you are uploading the theme .zip file, not the entire package you downloaded. In this case, you will be uploading "tenpo.zip".
(Video installation instructions below):

FTP Upload

To manually upload your new WordPress theme, login with your credentials to your website and locate the wp-content folder in your WordPress install files. Upload the un-zipped tenpo WordPress Theme folder into the: wp-content/themes folder.
Once uploaded, activate the theme by heading to the “Themes” menu in the WordPress Dashboard.

Installing Plugins

Once you have successfully installed and activated our theme, you have the option to manually install and activate plugins to enhance the functionality and features of your website. To do so, please follow the steps below:

  1. Go to the Tenpo section in your website's admin dashboard.
  2. Under the Tenpo menu, locate the "Required Plugins" option and click on it.
  3. You will be directed to a page where you can begin the process of installing the required plugins.
  4. Click on the "Begin installing plugins" button to initiate the installation process.
  5. Once the plugins have been installed, you can proceed to activate them by clicking on the "Activate" button next to each plugin.

Contact Form

- To create Contacts Form for Class, please do the following:
1. Go to Dashboard > Contact > Contact Forms > Add New.
2. Enter form name "Join Class".
3. In the tab "Form" you can use default form layout or enter this code lines:
4. Please cope this form shortcode and paste it into your post, page, or text widget content: 

						<div class="zrg-join-class-form">
						<h3 class="zrg-text-1">Find Answer? Ask Us</h3>
						<label>[text* your-name placeholder "Full name"]</label>
						<label>[text* your-phone placeholder "Phone number"]</label>
						<label>[email* your-email placeholder "Email address"]</label>
						<label>[textarea your-message placeholder "Message"]</label>
						<div class="zrg-join-class-button zrg-text-2">[submit "Submit"]</div>
						</div>
						[hidden class-title "CLASS_VALUE" ]

5. In the tab "Mail", enter this your email address.
6. Save Changes.

- To insert Contact form to page, please do the following:
1. Go to Dashboard > Pages > Contact Us. Click to "Edit with WPBakery Page Builder" button.
2. In WPBakery Page Builder widget "Author Block" in Form field, select your created CF7 form.

MC4WP.Mailchimp for WordPress

- To create Mailchimp Subscription Form, please do the following:
1. Please go to Dashboard > MC4WP > Mailchimp and enter Mailchimp API key in Mailchimp settings, and save changes. 2. Go to Dashboard > MC4WP > Forms > Edit Form.
3. Enter form name "Newsletter subscription".
4. In the tab "Form Code" enter this code lines:

					<input type="email" name="EMAIL" placeholder="Enter your email" required />
					<input type="submit" value="Subscribe now" />

4. In the tab "Messages", Please enter your custom messages
5. In the tab "Settings", you can set Mailchimp specific settings
6. In the tab "Appearance", you can select Form Style from dropzerodown select
7. Save Changes.

Update Our Themes

Manually Update (Appearance -> Themes)

You can update our theme via WordPress Panel, Go to Dashboard > Appearance > Themes. Click "Add New" -> "Upload Theme". Choose tenpo theme "tenpo.zip" file and select “Install Now”.

Manually Update (FTP or Cpanel)

You can update our theme using via FTP or Cpanel.
Download the latest version of the theme and you rename the folder contain the old theme and create a new folder and insert the latest theme in here.

- Uploads folder writable

- ZipArchive
- PHP Memory Limit: (256MB )
